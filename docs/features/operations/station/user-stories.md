## Epic 1: 站点基础信息

来源功能点：F001, F002, F003, F004, F005, F006, F007, F008

### US-001: 查看站点列表

**优先级：** [MVP]
**功能点：** F001
**角色：** 运营经理

> 作为运营经理，
> 我希望以列表形式查看所有站点的概要信息，
> 以便快速了解旗下所有站点的运营状态。

**验收标准：**

1. 假设已有多个站点数据，当进入站点管理页面，那么显示站点列表，包含站点名称、编码、地址、联系方式、状态列。
2. 假设站点数量超过单页显示数量，当翻页，那么正确显示对应页的站点数据。
3. 假设需要查找特定站点，当在搜索框输入站点名称或编码，那么列表实时过滤显示匹配的站点。
4. 假设需要按条件筛选，当选择"状态"或"区域"筛选条件，那么列表只显示符合条件的站点。

**关联：** US-002, US-003, US-004
**前置依赖：** 无

### US-002: 新增站点

**优先级：** [MVP]
**功能点：** F002
**角色：** 运营经理

> 作为运营经理，
> 我希望录入新站点的基本信息，
> 以便将新开业的站点纳入系统管理。

**验收标准：**

1. 假设在站点列表页，当点击"新增站点"按钮，那么打开新增站点表单页面。
2. 假设填写表单时，当选择"自动生成"站点编码，那么系统自动填入一个唯一编码。
3. 假设填写表单时，当选择"手工输入"站点编码并输入编码，那么系统校验编码唯一性并提示结果。
4. 假设完整填写站点名称、编码、地址、联系方式、营业时间，当点击"保存"，那么站点创建成功并跳转到站点列表，新站点出现在列表中。
5. 假设必填字段未填写完整，当点击"保存"，那么表单提示必填字段错误，不允许提交。
6. 假设需要设置站点地理位置，当在地址栏选择或输入地址，那么自动获取经纬度坐标；也支持在地图上手动标注。

**关联：** US-001
**前置依赖：** 无

---

### US-003: 编辑站点信息

**优先级：** [MVP]
**功能点：** F003
**角色：** 站长

> 作为站长，
> 我希望修改我负责站点的基本信息，
> 以便在站点信息发生变更时保持系统数据的准确性。

**验收标准：**

1. 假设在站点列表或详情页，当点击"编辑"按钮，那么打开站点编辑表单，表单预填当前站点信息。
2. 假设修改了站点名称、地址或联系方式等信息，当点击"保存"，那么站点信息更新成功，页面显示更新后的信息。
3. 假设站点编码字段，当尝试修改，那么该字段不可编辑（站点编码创建后不允许修改）。
4. 假设修改后未保存直接离开页面，当点击其他链接或返回，那么系统提示"有未保存的修改"确认弹窗。

**关联：** US-001, US-004
**前置依赖：** US-002

### US-004: 查看站点详情

**优先级：** [MVP]
**功能点：** F004
**角色：** 站长

> 作为站长，
> 我希望查看站点的完整详细信息，
> 以便全面了解站点的配置和运营状态。

**验收标准：**

1. 假设在站点列表页，当点击某站点的名称或"查看详情"按钮，那么进入站点详情页。
2. 假设进入站点详情页，当页面加载完成，那么显示站点基础信息、枪配置概览、班次概览、照片预览等信息区块。
3. 假设站点有关联的枪配置，当查看详情页，那么显示枪的数量统计和快速入口链接到枪配置列表。
4. 假设站点有上传的照片，当查看详情页，那么在照片区域显示主展示图的缩略图。

**关联：** US-001, US-003, US-015, US-021
**前置依赖：** US-002

### US-005: 删除站点（软删除）

**优先级：** [MVP+]
**功能点：** F005
**角色：** 运营经理

> 作为运营经理，
> 我希望将已停用的站点标记为停用状态，
> 以便在列表中隐藏不再运营的站点，同时保留历史数据。

**验收标准：**

1. 假设在站点列表或详情页，当点击"删除"按钮，那么弹出确认对话框，提示"删除后站点将标记为停用，历史数据将保留"。
2. 假设在确认对话框，当点击"确认删除"，那么站点状态变为"停用"，在默认列表视图中不再显示。
3. 假设站点下仍有进行中的业务（如未完成的订单），当尝试删除，那么系统提示"该站点存在未完成的业务，无法删除"。
4. 假设需要查看已停用的站点，当在筛选条件中选择"显示停用站点"，那么列表中显示已停用的站点（灰显或带停用标记）。

**关联：** US-001
**前置依赖：** US-002

### US-006: 多站点切换

**优先级：** [MVP]
**功能点：** F006
**角色：** 运营经理

> 作为运营经理，
> 我希望在系统中快速切换当前操作的站点上下文，
> 以便查看和管理不同站点的业务数据。

**验收标准：**

1. 假设用户有多个站点的访问权限，当查看页面顶部或侧边栏，那么显示"当前站点"选择器，显示当前站点名称。
2. 假设点击"当前站点"选择器，当下拉列表展开，那么显示用户有权访问的所有站点，支持搜索筛选。
3. 假设在下拉列表中选择另一个站点，当选择完成，那么系统切换到该站点上下文，页面数据刷新为新站点的数据。
4. 假设用户只有一个站点的权限，当查看页面，那么站点选择器显示该站点名称但不可切换（或不显示选择器）。

**关联：** 全局功能，影响所有模块
**前置依赖：** US-002

### US-008: 将站点归属到分组

**优先级：** [MVP+]
**功能点：** F007
**角色：** 运营经理

> 作为运营经理，
> 我希望将站点归属到指定的分组，
> 以便通过分组维度管理和筛选站点。

**验收标准：**

1. 假设已有站点和分组，当在站点编辑页面选择"所属分组"下拉，那么显示所有可用分组供选择。
2. 假设选择一个分组并保存，当保存成功，那么该站点归属到选定分组，在分组视图下可见。
3. 假设在分组管理页面，当选择某分组查看，那么显示该分组下所有站点列表。
4. 假设在站点列表页，当按分组筛选，那么只显示该分组下的站点。

**关联：** US-007, US-001
**前置依赖：** US-007

### US-009: 区域管理

**优先级：** [MVP+]
**功能点：** F008
**角色：** 运营经理

> 作为运营经理，
> 我希望管理区域层级结构并将站点归属到区域，
> 以便按地理维度进行站点的分级管理和数据统计。

**验收标准：**

1. 假设在区域管理页面，当点击"新建区域"，那么可以创建区域并设置上级区域（支持多级：如省→市→区）。
2. 假设已创建区域层级，当查看区域列表，那么以树形结构展示区域层级关系。
3. 假设在站点编辑页面，当选择"所属区域"，那么显示区域树供选择。
4. 假设站点已归属区域，当按区域筛选站点列表，那么显示该区域及下级区域的所有站点。
5. 假设需要修改区域名称或层级，当编辑区域并保存，那么区域信息更新，已归属该区域的站点关联关系保持。

**关联：** US-001, US-002
**前置依赖：** 无

**优先级：** [MVP]

**前置依赖：** US-002

### US-011: 新增枪配置

**优先级：** [MVP]
**功能点：** F010
**角色：** 设备管理员

> 作为设备管理员，
> 我希望为站点添加新的枪配置，
> 以便将新安装的加注设备纳入系统管理。

**验收标准：**

1. 假设在枪配置列表页，当点击"新增枪"，那么打开新增枪配置表单。
2. 假设选择燃料类型，当下拉选择，那么显示固定枚举选项（92#、95#、98#、柴油、CNG、LNG），另有"其他"选项支持自定义输入。
3. 假设填写枪号、选择燃料类型、设置初始单价，当点击"保存"，那么枪配置创建成功，出现在枪列表中，默认状态为"启用"。
4. 假设枪号在当前站点已存在，当尝试保存，那么系统提示"枪号已存在"。
5. 假设自动关联当前站点，当保存成功，那么枪与当前站点自动关联。

**关联：** US-010, US-014
**前置依赖：** US-002

### US-012: 编辑枪配置

**优先级：** [MVP]

2. 假设修改燃料类型或单价，当保存，那么枪配置更新成功，列表显示新信息。
3. 假设枪号字段，当尝试修改，那么系统不允许修改（枪号创建后不可变更）。

**关联：** US-010, US-011
**前置依赖：** US-011

### US-013: 设置枪单价

**优先级：** [MVP]
**功能点：** F012
**角色：** 站长

> 作为站长，
> 我希望设置各枪的销售单价，
> 以便根据运营策略调整燃料价格。

**验收标准：**

1. 假设在枪列表页，当点击某枪的"设置单价"或进入单独的"价格设置"入口，那么显示单价设置界面。
2. 假设输入新单价，当点击"保存"，那么枪单价更新成功，立即生效。
3. 假设选择多个相同燃料类型的枪，当批量设置单价并保存，那么所选枪的单价统一更新。
4. 假设输入无效单价（如负数或非数字），当尝试保存，那么系统提示输入错误。
5. 假设单价变更，当保存成功，那么系统记录变更操作日志（操作人、时间、变更前后价格）。

**关联：** US-010, US-012
**前置依赖：** US-011

### US-014: 查看枪实时状态

**优先级：** [MVP]
**功能点：** F013
**角色：** 站长

> 作为站长，
> 我希望查看各枪的实时状态，
> 以便监控设备运行情况和及时发现故障。

**验收标准：**

1. 假设枪已接入数据采集系统，当查看枪列表或详情，那么显示实时状态（空闲/充装中/离线/故障）。
2. 假设枪正在进行充装操作，当查看状态，那么显示"充装中"并可能显示当前充装量。
3. 假设枪状态变为异常（如离线超时、故障报警），当查看列表，那么该枪以醒目标记（如红色标签）提示。
4. 假设点击某枪查看详情，当进入详情页，那么显示更详细的设备连接信息和历史状态记录。

**关联：** US-010
**前置依赖：** US-011

3. 假设枪已有历史充装记录，当修改所属站点，那么系统提示确认，历史数据仍关联原站点。

**关联：** US-011, US-006
**前置依赖：** US-002, US-011

### US-016: 枪启用/停用

**优先级：** [MVP+]
**功能点：** F015
**角色：** 站长

> 作为站长，
> 我希望启用或停用指定枪的使用状态，
> 以便在设备维护或故障时暂停该枪的业务操作。

**验收标准：**

1. 假设在枪列表，当选择一个启用状态的枪并点击"停用"，那么弹出确认对话框。
2. 假设确认停用，当操作完成，那么枪状态变为"停用"，该枪不再参与充装业务。
3. 假设在枪列表，当选择一个停用状态的枪并点击"启用"，那么枪状态变为"启用"，可继续参与业务。
4. 假设查看枪列表，当有停用的枪，那么停用枪以灰色或特殊标记显示。

**关联：** US-010
**前置依赖：** US-011

## Epic 3: 站点照片/图片

**优先级：** [MVP+]

**角色：** 站长

### US-019: 删除站点照片

**优先级：** [MVP+]
**功能点：** F023
**角色：** 站长

> 作为站长，
