## 3. 全局组件设计

### G01: 站点选择器 (StationSelector)

**位置：** 顶部导航栏右侧，固定展示
**组件：** Select + Dropdown 组合

| 区域 | 内容 | 说明 |
|------|------|------|
| 触发器 | 当前站点名称 + 下拉箭头 | 点击展开 |
| 下拉-搜索 | Input.Search | 自动聚焦，支持站点名称/编码搜索 |
| 下拉-最近访问 | 站点列表(最多5个) | 带收藏星标 |
| 下拉-全部站点 | 分区域分组显示 | 可展开/折叠 |

**交互行为：**
| 触发 | 行为 |
|------|------|
| 点击选择器 | 展开下拉，自动聚焦搜索框 |
| 输入搜索 | 实时过滤站点列表 |
| 选择站点 | 切换当前站点上下文，页面数据刷新，URL不变 |
| 快捷键 Ctrl/Cmd+K | 全局呼出站点搜索 |

## 4. 页面详细设计

### P01: 站点列表页 (StationList)

**路由：** `/operations/station`
**对应 Story：** US-001, US-005
**权限：** `ops_manager`, `station_master`, `finance`

#### 页面头部

- **面包屑：** 首页 / 基础运营 / 站点管理
- **标题：** 站点管理
- **操作按钮：** 
  - `[+ 新增站点]` (Primary) → 跳转 P03
  - `[导出]` (Default) → 导出当前筛选结果

#### 视图切换

| 视图 | 图标 | 说明 | 适用场景 |
|------|------|------|---------|
| 表格视图 | TableOutlined | 数据密集，适合批量操作 | 桌面端默认 |
| 卡片视图 | AppstoreOutlined | 状态直观，适合快速浏览 | 移动端默认 |

#### 筛选区域

| 字段 | 组件 | 数据源 | 默认值 | 宽度 |
|------|------|--------|--------|------|
| 关键词 | Input.Search | - | - | 240px |
| 状态 | Select | `active`/`inactive`/`all` | `active` | 120px |
| 区域 | TreeSelect | Region API | 全部 | 180px |
| 分组 | Select | StationGroup API | 全部 | 150px |

**高级筛选(展开)：**
| 字段 | 组件 | 数据源 |
|------|------|--------|
| 创建时间 | DatePicker.RangePicker | - |

#### 数据表格 (表格视图)

| 列名 | 字段 | 宽度 | 对齐 | 渲染 | 排序 |
|------|------|------|------|------|------|
| 站点名称 | `name` | 180px | left | 链接→P02，带主图缩略图 | ✅ |
| 站点编码 | `code` | 100px | left | 文本 | ✅ |
| 状态 | `status` | 80px | center | Tag (active=绿, inactive=灰) | ✅ |
| 地址 | `address` | 200px | left | 文本，超长省略 | - |
| 联系方式 | `contactPhone` | 120px | left | 文本 | - |
| 枪数量 | `nozzleCount` | 80px | center | 数字 | ✅ |
| 区域 | `region.name` | 120px | left | 文本 | - |
| 分组 | `group.name` | 100px | left | Tag | - |
| 更新时间 | `updatedAt` | 150px | left | 日期 YYYY-MM-DD HH:mm | ✅ |
| 操作 | - | 150px | center | [查看] [编辑] [更多▼] | - |

**操作列详情：**
| 操作 | 类型 | 权限 | 行为 |
|------|------|------|------|
| 查看 | Link | all | 跳转 P02 |
| 编辑 | Link | `station:update` | 跳转 P03 |
| 更多-停用 | Dropdown | `station:delete` | Popconfirm → 软删除 |
| 更多-恢复 | Dropdown | `station:delete` | 仅停用状态显示 |

**列表排序：** 默认按"异常状态置顶 + 更新时间倒序"
**分页：** 默认 20 条/页，可选 10/20/50/100

#### 卡片视图 (StationCard)

```
┌──────────────────────────────────────┐
│ [主图]                               │
├──────────────────────────────────────┤
│ 站点名称                    [状态Tag] │
│ 📍 地址信息...                       │
│ 📞 联系电话                          │
├──────────────────────────────────────┤
│ 🔫 枪数: 6  |  👥 员工: 12           │
└──────────────────────────────────────┘
点击整卡 → 进入详情页
```

**卡片布局：** 
- 桌面端：每行 4 卡片
- 平板端：每行 2-3 卡片
- 移动端：每行 1 卡片

#### 异常提示区

当有异常站点时，在筛选区上方显示警告条：
```
⚠️ 有 2 个站点存在异常：城北加气站(3台枪故障)、城南加气站(已停用)  [查看详情]
```

#### 交互行为

| 触发 | 行为 | 说明 |
|------|------|------|
| 点击 [新增站点] | 跳转 P03 | 新增模式 |
| 点击站点名称/卡片 | 跳转 P02 | 查看详情 |
| 点击 [编辑] | 跳转 P03 | 编辑模式，预填数据 |
| 点击 [停用] | Popconfirm | 提示关联数据数量，确认后软删除 |
| 切换视图 | 本地存储 | 记住用户偏好 |
| 搜索/筛选 | 即时刷新 | 重置分页到第1页 |

#### 空状态

```
还没有站点信息，添加第一个站点开始管理
[+ 新增站点]
```

#### 响应式适配

| 断点 | 布局调整 |
|------|---------|
| ≥1200px | 表格视图完整列 |
| 768-1199px | 隐藏次要列(区域、分组、更新时间) |
| <768px | 强制卡片视图，底部固定新增按钮 |

### P03: 站点表单页 (StationForm)

**路由：** `/operations/station/create`, `/operations/station/:id/edit`
**对应 Story：** US-002, US-003
**权限：** `ops_manager` (新增), `station_master` (编辑)

#### 页面头部

- **面包屑：** 首页 / 基础运营 / 站点管理 / {新增站点|编辑站点}
- **标题：** 新增站点 / 编辑站点
- **操作按钮：**
  - `[保存]` (Primary)
  - `[取消]` → 返回列表/详情

#### 表单布局: 分步表单 (Steps)

| 步骤 | 标题 | 内容 |
|------|------|------|
| 1 | 基础信息 | 站点名称、编码、状态 |
| 2 | 位置信息 | 地址、坐标、区域分组 |
| 3 | 联系与营业 | 联系方式、营业时间 |
| 4 | 确认提交 | 信息预览 |

---

#### 步骤1: 基础信息

| 字段 | 标签 | 组件 | 必填 | 校验规则 | 默认值 | 说明 |
|------|------|------|------|---------|--------|------|
| `codeMode` | 编码方式 | Radio.Group | 是 | - | `auto` | `auto`=自动生成, `manual`=手工输入 |
| `code` | 站点编码 | Input | 条件 | 2-32字符, 字母数字下划线 | (自动生成) | `codeMode=manual`时显示 |
| `name` | 站点名称 | Input | 是 | 2-100字符 | - | - |
| `status` | 状态 | Select | 是 | - | `active` | 编辑时可修改 |

**编码自动生成逻辑：**
- 格式: `ST{年份后2位}{月}{日}{4位序号}`
- 示例: `ST260215001`

**字段联动：**
- 编码方式选择"自动生成"时，编码字段只读并显示预览值
- 编码方式选择"手工输入"时，编码字段可编辑

**编辑模式限制：**
- `code` 字段不可编辑，显示为只读 + 帮助文字

---

#### 步骤2: 位置信息

| 字段 | 标签 | 组件 | 必填 | 校验规则 | 默认值 |
|------|------|------|------|---------|--------|
| `address` | 详细地址 | Input.TextArea | 是 | 5-255字符 | - |
| `mapPicker` | 地图选点 | MapPicker (自定义) | 否 | - | - |
| `latitude` | 纬度 | InputNumber | 否 | -90~90, 7位小数 | (地图选点自动填充) |
| `longitude` | 经度 | InputNumber | 否 | -180~180, 7位小数 | (地图选点自动填充) |
| `regionId` | 所属区域 | TreeSelect | 否 | - | (上次选择) |
| `groupId` | 所属分组 | Select | 否 | - | - |

**地图选点组件：**
- 支持地址搜索定位
- 支持地图点击标注
- 点击后自动填充经纬度字段

---

#### 步骤3: 联系与营业

| 字段 | 标签 | 组件 | 必填 | 校验规则 | 默认值 |
|------|------|------|------|---------|--------|
| `contactName` | 联系人 | Input | 否 | 2-50字符 | - |
| `contactPhone` | 联系电话 | Input | 否 | 手机号/座机格式 | - |
| `businessHours.weekday` | 工作日营业时间 | TimeRangePicker | 否 | - | 06:00-22:00 |
| `businessHours.weekend` | 周末营业时间 | TimeRangePicker | 否 | - | 07:00-21:00 |
| `employeeSyncMode` | 员工管理模式 | Radio.Group | 是 | - | `sync` |

**员工管理模式说明：**
- `sync`: 从用户管理模块同步，不可在站点模块直接增删员工
- `local`: 站点独立维护员工信息

---

#### 步骤4: 确认提交

以 Descriptions 组件展示所有已填信息，用户确认后提交。

---

#### 表单行为

| 触发 | 行为 |
|------|------|
| 下一步 | 校验当前步骤 → 进入下一步 |
| 上一步 | 保留数据 → 返回上一步 |
| 保存(最后一步) | 全量校验 → 提交 → 成功后跳转详情页/列表页 |

## 7. 表单校验规则汇总

### 站点表单

| 字段 | 规则 |
|------|------|
| `code` | 必填(manual模式)；2-32字符；仅字母数字下划线；唯一校验(实时远程) |
| `name` | 必填；2-100字符 |
| `address` | 必填；5-255字符 |
| `latitude` | 选填；-90到90；最多7位小数 |
| `longitude` | 选填；-180到180；最多7位小数 |
| `contactPhone` | 选填；手机号(11位)或座机(区号-号码)格式 |
| `contactName` | 选填；2-50字符 |

### 枪配置表单

| 字段 | 规则 |
|------|------|
| `nozzleNo` | 必填；1-20字符；站点内唯一(实时远程) |
| `fuelTypeId` | 必填 |
| `unitPrice` | 必填；0.01-99.99；最多2位小数 |
| `dispenserNo` | 选填；1-20字符 |

### 班次表单

| 字段 | 规则 |
|------|------|
| `name` | 必填；2-50字符；站点内唯一 |
| `startTime` | 必填 |
| `endTime` | 必填 |

### 排班表单

| 字段 | 规则 |
|------|------|
| `scheduleDate` | 必填；不能早于今天(历史日期只读) |
| `shiftId` | 必填 |
| `employeeId` | 必填 |
