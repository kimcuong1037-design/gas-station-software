## 1. 模块概述

### 1.1 业务目标

巡检/安检管理是加气站安全运营的核心保障模块，确保：
- 制定覆盖全站各区域和设备的定期安全巡检计划
- 巡检任务的分配、执行和结果记录全流程闭环管理
- 巡检中发现的问题及时登记、分配、整改、验收闭环
- 巡检数据的统计分析、日报生成和合规报表输出

### 1.2 模块边界

| 范围 | 说明 |
|------|------|
| **包含** | 安检计划管理（按日/周/月生成计划、关联机构与任务）、安检任务管理（任务下发/分配/状态跟踪）、检查项目配置（项目分类/检查标准/模板）、标签管理（标签分类/排序）、巡检执行与日志（执行记录/检查结果/照片）、巡检日报（按日汇总/完成率/异常统计）、站点执行报表（站点维度统计/整改进度）、问题记录（登记/分配/整改/闭环）、安检统计（多维度趋势分析）、检查报表（合规报表/导出） |
| **不包含** | 设备台账管理（属于设备设施管理 1.3）、维保工单管理（属于设备设施管理 1.3，但巡检问题可触发联动）、视频监控巡检（属于远期智慧全景平台）、移动端巡检执行界面（属于移动端开发阶段 5+） |

### 1.3 与其他模块关系

| 关联模块 | 关系说明 |
|----------|----------|
| 站点管理 (1.1) | 巡检计划关联站点；执行人来源于站点的员工与排班数据 |
| 交接班管理 (1.2) | 交接班时可查看当日巡检完成情况 |
| 设备设施管理 (1.3) | 检查项目与设备台账关联（如"x# 加气机"）；巡检问题可触发维保工单（F018/F032 of 1.3） |
| 风控中心 (5.4) | 严重巡检问题可同步至风控中心 |
| 数据分析 (7.1) | 巡检统计数据为经营看板和多维分析的数据源 |

## 3. 依赖关系

### 3.1 模块内依赖

- F002（创建安检计划）→ F003, F004, F005: 需先有计划才能编辑、查看详情、删除
- F002（创建安检计划）→ F008（下发安检任务）: 安检任务从安检计划中生成
- F015（新增检查项目）→ F024（记录巡检执行）: 执行巡检时需要已配置的检查项目
- F008（下发安检任务）→ F011（更新任务状态）: 任务下发后才有状态流转
- F008（下发安检任务）→ F024（记录巡检执行）: 任务下发后才能执行巡检
- F024（记录巡检执行）→ F037（登记问题）: 巡检执行中发现异常时登记问题
- F037（登记问题）→ F038, F039, F040: 问题登记后进入处理/整改/闭环流程
- F024（记录巡检执行）→ F028-F029（巡检日报）: 日报数据依赖执行记录
- F024, F037 → F031-F033（站点执行报表）: 报表依赖巡检执行和问题数据
- F024, F037 → F045-F047（安检统计）: 统计依赖巡检执行和问题数据

### 3.2 跨模块依赖

| 功能点 | 依赖模块 | 依赖说明 |
|--------|----------|----------|
| F001-F005（安检计划） | 站点管理 (1.1) | 计划关联站点，需获取站点列表 |
| F009（分配执行人） | 站点管理 (1.1) | 执行人来源于站点的员工列表 |
| F015（新增检查项目） | 设备设施管理 (1.3) | 检查项目可关联设备台账（如"x# 加气机"），引用设备编号 |
