## 2. 页面详细设计

### 2.1 P01 报表总览

**布局：** 上下分区布局

**区域一：快速入口（顶部）**
- 5 张卡片横向排列，对应 5 种标准报表类型
- 每张卡片包含：图标 + 报表类型名称 + 最近生成时间
- 点击跳转到 P02 对应 Tab
- 卡片样式：与 7.1 KPICard 风格一致

**区域二：收藏报表 + 最近查看（中部，左右两栏）**
- 左栏"我的收藏"：收藏报表列表，点击直接查看详情
- 右栏"最近查看"：最近 5 条记录，含标题 + 类型标签 + 生成时间

**区域三：报表日历（底部）**
- Ant Design Calendar 组件
- 每个日期格上渲染彩色圆点标记（按报表类型着色）
- 点击日期弹出 Popover 展示当日报表列表
- 日历上方可切换月份

**端到端走查（US-006）：**
1. 用户进入报表总览 → 看到快速入口 + 收藏 + 日历
2. 点击日历上某个有标记的日期 → Popover 展示报表列表
3. 点击列表中的报表 → 跳转到 P02 查看详情
4. ✅ 每一步均有明确的交互目标

### 2.2 P02 标准报表

**布局：** Tab 切换 + 内容区

**Tab 栏（顶部）：**
- 5 个 Tab：经营日报 | 经营月报 | 交接班报表 | 巡检报表 | 库存报表
- Tab 切换时内容区刷新

**筛选条件栏（Tab 下方）：**
- 日期选择器（日报用 DatePicker，月报用 MonthPicker，巡检/库存用 RangePicker）
- 站点选择器（Select 多选）
- "生成报表"按钮（Primary）
- "导出"按钮 + "收藏"按钮

**报表内容区：**

*经营日报/月报：*
- 顶部 KPI 汇总：4 张统计卡片（营业额/订单/充装量/客单价）+ 增长标识
- 表格：按站点分列的明细数据
- 图表：燃料类型占比饼图 + 站点对比柱状图
- 月报额外：日均趋势折线图

*交接班报表：*
- 表格：各班次汇总（营收/订单/充装量/现金/在线/刷卡）
- 图表：支付方式分布饼图 + 班次对比柱状图

*巡检报表：*
- 顶部汇总：计划完成率/问题数/整改率 3 张卡片
- 表格：按区域的问题分布
- 高亮区：未整改的高/紧急级别问题列表（红色标识）

*库存报表：*
- 表格：按站点+燃料类型的进出库明细
- 异常损耗行红色高亮
- 图表：罐容比趋势折线图

**端到端走查（US-001 经营日报）：**
1. 用户进入 P02 → 默认选中"经营日报"Tab
2. 选择日期（默认今天）→ 点击"生成报表"
3. 内容区展示 KPI 卡片 + 站点明细表 + 图表
4. 点击"导出 Excel" → Toast 提示 MVP 占位
5. 点击收藏星标 → 星标高亮，报表加入收藏
6. ✅ 完整用户旅程可达

### 2.3 P03 自定义报表

**布局：** 左右分栏（模板列表 + 操作区）

**左侧面板（模板列表，宽 280px）：**
- 搜索框（模板名称搜索）
- 分组展示：系统模板（不可编辑标记）/ 自定义模板
- 每个模板项：名称 + 数据源标签 + 上次使用时间
- 底部"新建模板"按钮
- 选中模板高亮

**右侧操作区（模板未选中时）：**
- 空状态引导：选择左侧模板或创建新模板

**右侧操作区（模板选中后）：**
- 顶部：模板名称 + 描述 + 操作按钮（编辑/删除/生成报表）
- 模板配置摘要：数据源、维度、指标（Tag 展示）
- 时间范围选择器 + "生成报表"按钮
- 报表预览区：
  - Toggle 切换：表格模式 / 图表模式
  - 表格模式：标准 Ant Design Table
  - 图表模式：根据维度自动选择合适图表类型
  - 底部操作栏：导出 Excel / 导出 PDF

**模板构建器（Drawer 弹出，新建/编辑时）：**
- Steps 组件，4 步：
  - Step 1：数据源选择（3 个 Radio Card：经营数据/库存数据/巡检数据）
  - Step 2：维度选择（Checkbox Group，选项根据数据源动态变化）
  - Step 3：指标选择（Checkbox Group，选项根据数据源动态变化）
  - Step 4：默认筛选 + 命名（站点范围、时间范围、模板名称、描述、标签）
- 每步有"上一步"/"下一步"/"保存"按钮

**端到端走查（US-007 创建模板 → US-009 生成报表）：**
1. 用户进入 P03 → 左侧模板列表
2. 点击"新建模板" → Drawer 打开模板构建器
3. Step 1 选择"经营数据" → Step 2 选择"时间+站点" → Step 3 选择"营业额+订单数" → Step 4 命名"月度站点对比"
4. 保存 → 模板出现在左侧列表
5. 选中该模板 → 右侧显示配置摘要
6. 选择时间范围 → 点击"生成报表"
7. 预览区展示数据表格
8. 切换到图表模式 → 展示柱状图
9. 点击"导出 Excel" → Toast 提示 MVP 占位
10. ✅ 完整创建→生成→预览→导出旅程可达

## 3. 交互规范

### 3.1 全局交互

| 交互 | 行为 |
|------|------|
| 快速入口卡片 hover | 轻微上浮阴影 + cursor:pointer → onClick 跳转 P02 |
| 日历日期点击 | 有标记的日期弹出 Popover；无标记日期无响应 |
| 收藏星标点击 | 切换收藏状态，星标 filled/outlined 切换 + Toast 反馈 |
| 报表类型标签 | 不同类型用不同颜色 Tag（经营=蓝/巡检=橙/库存=绿/交接班=紫/自定义=灰） |
| 模板删除 | 弹出 Popconfirm 二次确认 |
| 导出按钮 | MVP 阶段点击后 message.info 提示，按钮保持可用态 |

### 3.2 空状态

| 场景 | 展示 |
|------|------|
| 无收藏报表 | "暂无收藏报表，可在报表详情中点击星标收藏" |
| 无最近查看 | "暂无查看记录" |
| 日历当月无报表 | 日历正常展示，无圆点标记 |
| 自定义模板为空 | "暂无自定义模板，点击下方按钮创建" + 新建按钮 |
| 报表预览无数据 | "请选择模板并设置时间范围后生成报表" |

### 3.3 响应式

- 报表总览：快速入口卡片 Responsive Grid（xl=5列, lg=3列, md=2列）
- 自定义报表：左侧面板在 md 以下折叠为 Drawer 触发
- 表格：scroll.x 适配，确保横向可滚动
