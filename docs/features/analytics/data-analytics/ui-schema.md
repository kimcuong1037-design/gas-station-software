```text

└── 数据分析（Sub-group，二级，不可点击，灰色分组标题）
```

### P01: 经营看板（BusinessDashboard）

**路由：** `/analytics/data-analytics/dashboard`
**对应 Story：** US-001, US-002, US-003, US-004, US-005, US-006（MVP+）
**权限：** 管理员 / 运营经理（全站） / 站长（仅本站）/ 财务人员

#### 页面头部

- **面包屑：** 数据分析与报表 / 数据分析 / 经营看板
- **标题：** 经营看板
- **右侧操作：** 无（纯只读，无操作按钮）
- **RequirementTag：** 放置在页面标题右侧

#### 全局筛选栏（固定在内容区顶部，白底卡片）

| 字段 | 组件 | 数据源 | 默认值 | 说明 |
|------|------|--------|--------|------|
| 时间粒度 | `Radio.Group`（按钮模式） | 枚举：日 / 周 / 月 / 年 | 日（昨日） | 切换立即生效，无需确认 |
| 站点 | `Select`（多选，`mode="multiple"`） | 站点列表 API | 全部站点 | 站长角色只显示其负责站点 |
| 数据截至时间 | 灰色文字显示 `数据截至 HH:MM` | 页面加载时间 | — | 只读，放于筛选栏右侧 |

#### 区域一：KPI 卡片行（US-001）

布局：`<Row gutter={[16,16]}>` 内 4 个 `<Col span={6}>`，共 4 张卡片

每张 KPI 卡片（组件：`KPICard.tsx`）包含：

| 元素 | 内容 | 说明 |
|------|------|------|
| 标题 | 营业额 / 订单数 / 充装量 / 客单价 | 灰色小字 |
| 主数值 | `<Statistic value={...} />` | 大字号，保留 2 位小数（金额）或整数（订单数） |
| 同比变化 | `<GrowthBadge />` 自定义组件 | 上涨：绿色箭头↑ + 百分比；下跌：红色箭头↓ + 百分比；无数据：`—` |
| 迷你趋势图 | `<MiniChart />` 已有组件 | 宽度 100%，高度 40px，展示近 7 天走势 |
| 异常高亮规则 | 变化 ≥ ±20%：卡片背景色变为浅红/浅绿 | 变化 ≥ ±10%：主数值加粗 |

**4 张卡片定义：**

| # | 标题 | 数据字段 | 格式 | 同比字段 |
|---|------|----------|------|---------|
| 1 | 营业额 | `kpi.totalRevenue` | `¥{n}` | `kpi.revenueGrowth` |
| 2 | 订单数 | `kpi.totalOrders` | `{n} 单` | `kpi.ordersGrowth` |
| 3 | 充装量 | `kpi.totalFuelVolume` | `{n} m³` | `kpi.fuelVolumeGrowth` |
| 4 | 客单价 | `kpi.avgOrderValue` | `¥{n}` | `kpi.avgOrderValueGrowth` |

**Tooltip 说明：**
- 客单价卡片右上角 `<Tooltip title="总营业额 ÷ 总订单数" />` 问号图标
- 同比字段右侧 `<Tooltip title="与去年同期相比的增长百分比" />` 问号图标

#### 区域二：销售趋势折线图（US-002）

布局：`<Row gutter={16}>` 内左侧 `<Col span={16}>` 放折线图，右侧 `<Col span={8}>` 放燃料占比环形图

**销售趋势折线图（左）**

| 元素 | 内容 |
|------|------|
| 组件 | `<LineChart />` 已有组件（ECharts） |
| 卡片标题 | 销售趋势 |
| X 轴 | 时间（日期/周/月，随时间粒度筛选变化） |
| Y 轴 | 营业额（元） |
| 系列 | 单系列折线（营业额），可叠加显示充装量（辅助 Y 轴，虚线） |
| 图表高度 | 280px |
| Tooltip | 显示：时间 + 营业额 + 充装量 + 同比变化 |
| 空数据处理 | 缺少数据的点显示虚线连接；图表区域无数据时显示 `该时段暂无营业数据` |
| 加载状态 | 骨架屏（`<Skeleton active />`，高度 280px） |

**燃料类型销售占比环形图（右，US-003）**

| 元素 | 内容 |
|------|------|
| 组件 | `<PieChart />` 新增组件（ECharts `doughnut` 模式） |
| 卡片标题 | 油品销售占比 |
| 数据字段 | `fuelBreakdown[]` |
| 环形图配置 | `innerRadius: 50%`，中心显示总营业额 |
| 图例 | 底部图例，显示燃料类型名称 + 百分比 |
| Tooltip | 燃料名称 + 销售额（¥） + 占比（%） |
| 占比 < 5% | 合并为"其他"类别 |
| 图表高度 | 280px |

#### 区域三：站点 KPI 排行（US-004）

| 元素 | 内容 |
|------|------|
| 组件 | `<BarChart />` 新增组件（ECharts 横向柱状图） |
| 卡片标题 | 站点排行 |
| 右侧操作 | 指标切换 `<Select>` 组件：营业额 / 订单数 / 充装量 / 客单价（默认：营业额） |
| Y 轴 | 站点名称 |
| X 轴 | 选中指标值 |
| 显示条数 | 默认 Top 5，`limit=5` |
| 每行交互 | `cursor: pointer`，点击跳转到 `/analytics/data-analytics/sales?stationId={id}`（带站点参数） |
| 每行末尾 | 排名变化箭头（↑↓ 与上期对比） |
| 图表高度 | 200px |

#### 区域四：会员统计概览（US-006，MVP+）

布局：独立 `<Card>` 行，灰色背景与上方核心 KPI 视觉区分

4 张迷你统计卡片（`<Col span={6}>`）：

| 卡片 | 数据字段 | 格式 |
|------|----------|------|
| 会员总数 | `memberStats.totalMembers` | `{n} 人` |
| 本期新增 | `memberStats.newMembersThisPeriod` | `{n} 人` |
| 活跃会员 | `memberStats.activeMembers` | `{n} 人` |
| 会员消费占比 | `memberStats.memberRevenueRatio` | `{n}%` |

卡片上方显示浅蓝色 `<Alert type="info">` Banner：`ℹ 会员数据接入会员管理模块（Phase 4）后自动更新，当前为演示数据`

#### 交互行为

| 触发 | 行为 | 路由/说明 |
|------|------|---------|
| 切换时间粒度（日/周/月/年） | 全页数据刷新（KPI + 趋势 + 占比 + 排行） | 无路由跳转，本页更新 |
| 切换站点筛选 | 全页数据刷新 | 无路由跳转 |
| 点击站点排行某行 | 跳转到多维分析，自动代入该站点 | `/analytics/data-analytics/sales?stationId={stationId}` |
| hover KPI 卡片 | 显示"查看详情"链接（跳转多维分析） | `/analytics/data-analytics/sales` |
| 切换排行指标（Select） | 排行图数据更新 | 本页更新，无路由跳转 |

**路由：** `/analytics/data-analytics/sales`
**支持查询参数：** `?stationId={id}`（从经营看板跳转时自动代入）
**对应 Story：** US-007, US-008, US-009, US-010, US-011, US-012（MVP+）, US-013（MVP+）
**权限：** 管理员 / 运营经理 / 站长（仅本站）/ 财务人员

#### 页面头部

- **面包屑：** 数据分析与报表 / 数据分析 / 多维分析
- **标题：** 多维分析
- **右侧操作：** `[导出]` 按钮（`disabled`，`<Tooltip title="即将上线" />`，灰色）
- **RequirementTag：** 放置在页面标题右侧

#### 全局筛选面板（固定在内容区顶部，白底卡片，切换维度时筛选条件保持不变）

| 字段 | 组件 | 数据源 | 默认值 | 说明 |
|------|------|--------|--------|------|
| 时间范围 | `DatePicker.RangePicker` | — | 近 30 天 | 禁用今天之后的日期；超过 12 个月时 `<Alert>` 警告 |
| 站点 | `Select`（多选，最多 5 个） | 站点列表 | 全部 | 超出 5 个时提示"最多选择 5 个站点进行对比" |
| 燃料类型 | `Select`（多选） | 燃料类型枚举 | 全部 | 可多选筛选品类 |
| 对比方式 | `Radio.Group`（按钮模式） | 枚举：同比 / 环比 / 不对比 | 同比 | 影响表格同比/环比列的显示 |

#### 维度 Tab（US-007~US-010）

组件：`<Tabs>` 放在筛选面板下方

| Tab Key | 标签 | 对应 Story |
|---------|------|-----------|
| `time` | 时间分析 | US-007 |
| `station` | 站点对比 | US-008 |
| `fuel` | 品类分析 | US-009 |
| `segment` | 时段分析 | US-010 |

默认激活：`time`（时间分析）

切换 Tab 时：
- 筛选面板参数（时间范围、站点、燃料类型）**保持不变**
- 仅图表类型和表格列定义切换
