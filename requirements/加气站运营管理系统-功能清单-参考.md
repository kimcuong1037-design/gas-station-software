## 一、系统架构总览

### 1.1 产品体系

全国独家研发7套经营管理类子系统,全覆盖能源零售赛道:

| 子系统 | 定位 | 核心能力 |
|------|------|---------|
| **智慧零管系统** | 核心管理平台 | 站点、人员、财务、活动、设备等全流程管理 |
| **私域会员系统** | 会员运营 | 会员档案、等级权益、积分、储值卡、微商城 |
| **聚合支付系统** | 多渠道支付 | 线上线下支付、智能收银、手持POS、自动对账 |
| **精准营销系统** | 营销活动管理 | 电子券、充值活动、跨界合作、会员分级营销 |
| **纸电发票一体化系统** | 财务税务 | 纸质/电子发票管理、自动开票、对账 |
| **数字化运营系统** | 数据分析决策 | 可视化大屏、多维分析、定制报表 |
| **便利店系统** | 附加业务 | 商品管理、收银促销、库存管理 |

### 1.2 系统端架构

| 端 | 说明 | 典型用户 | 核心场景 |
|----|------|----------|----------|
| **管理后台** | Web 端，多级导航（顶部业务模块 + 左侧功能菜单），支持多标签页 | 运营、财务、管理层、巡检/安检人员 | 经营分析、活动配置、会员管理、报表导出 |
| **站端 POS** | 站点现场收银与监控，实时枪状态、充装记录、支付、交接班 | 收银员、站长、班组长、加油员 | 充装收银、会员识别、班次管理、现场监控 |
| **移动端** | 微信小程序/公众号/App | 普通会员、企业车队 | 找站、下单、支付、积分、充值、订单查询 |

### 1.3 业务场景定位

- 方便站点管理人员进行汽(油)品、气品、商品、人员、财务、活动等多项管理
- 支持电子券、异业核销和多种支付方式
- 系统自动生成财务报表,一键导出等功能
- 经营数据清晰查看,运营活动一键生成,站点智能管控
- 进销存数据移动可视化，库存、交易、班次、交接班等实时同步
- 全程不下车、车牌/人脸无感支付，提升客户体验与通行效率

## 二、功能清单（按业务域）

### 1. 基础运营

#### 1.1 站点管理

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **站点基础信息** | 站点名称、编码、地址（经纬度）、联系方式、营业时间 | 支持多站点，站端可显示当前站点（如「畅加测试站」）；多站点统一管理(连锁模式)；站点分组/区域管理 |
| **加注机/枪配置** | 枪号与燃料类型绑定（92#、95#、98#、柴油、CNG、LNG）、单价配置、枪与站点关联 | 站端可展示「实时充装枪状态」列表（枪号、燃料、金额、数量、单价、连接状态）；支持加油机/加气机数据采集与状态监控 |
| **充电桩映射** | 若为综合能源站，充电桩与站点、枪位或独立设备关联 | 与「充电系统」模块联动，充电桩状态监控、计费规则、订单与支付 |
| **站点照片/图片** | 站点形象展示、环境照片 | 用于App/小程序展示、监管报备 |
| **员工与排班** | 班次定义（如 1 天 1 班、多班）、负责人、营业时间、排班表 | 站端底部展示：班次、负责人、营业时间、交接班入口；员工排班管理、考勤管理 |
| **责任站点/维护站点** | 站点作为「责任站点」「维护站点」参与筛选与报表 | 大客户管理中「责任站点」筛选、企业维护站点归属 |

#### 1.2 交接班管理

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **班次累计统计** | 当前班次营业额、单数、充装量、收银方式汇总 | 站端底部「当前班次累计」，支持脱敏显示（*****）；按站点/班次/支付方式的收支汇总 |
| **交接班操作** | 交接班触发、班次切换、下一班登录、交接确认 | 站端「交接班」按钮；与收银员权限、班次负责人关联 |
| **现金解缴** | 现金收款记录、解缴登记、与班次关联、长短款记录 | 与财务、对账衔接；现金盘点功能 |
| **交接班异常** | 异常标注、未结订单提醒、长短款提示 | 可选：交接班前强制平账或确认；异常原因记录与审计 |
| **交接班报表** | 班次收支明细、交接单打印、历史交接记录查询 | 支持导出与审计 |

#### 1.3 设备设施管理（后台顶部一级模块）

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **设施监控** | 储罐液位/压力、加气机状态、传感器数据、设备健康度 | 可扩展：阈值告警、维保工单自动触发；油罐液位监控、安全预警 |
| **设备台账** | 设备档案、型号、安装日期、维保周期、设备照片 | 与巡检「检查项目」中的设备对应（如 x# 加油机、罐区）；设备编号唯一性管理 |
| **维保与工单** | 保养计划、维修记录、故障报修与闭环、维保提醒 | 与巡检「问题记录」联动；维保记录溯源 |
| **设备联网对接** | 加油机/加气机数据采集、油枪信息读取、实时状态监控 | 支持多种设备协议；数据实时同步 |

#### 1.4 巡检/安检管理（截图：巡检日志）

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **安检计划** | 按日/周/月生成巡检计划，可关联机构与任务 | 示例：「【01月26日】2026年1月巡检计划的任务」；支持周期性自动生成 |
| **安检任务** | 任务下发、执行人分配、任务状态（待执行/已完成）、任务提醒 | 与「执行人员」查询对应；支持移动端任务接收与执行 |
| **检查项目** | 项目配置（罐区环保、x# 加油机、配电室、加油区域、非油管理、设备管理等） | 支持按项目筛选，检查结果：正常/异常；检查项模板配置 |
| **标签管理** | 检查项标签、分类、排序 | 便于归类与报表；标签体系配置 |
| **巡检执行与日志** | 执行时间、执行人、机构、检查项、检查结果（●正常/异常）、现场照片 | 列表字段：序号、执行人员、机构名称、检查任务、检查项目、执行时间、检查结果、操作（详情）；支持移动端拍照上传 |
| **巡检日报** | 按日汇总巡检执行情况、完成率、异常项统计 | 与「站点执行报表统计」配合；支持定时推送 |
| **站点执行报表统计** | 按站点维度统计巡检完成率、异常项、整改进度 | 支持导出与看板；多维度对比分析 |
| **问题记录** | 巡检中发现的问题登记、分配、整改、闭环、整改照片 | 与设备维保、风控可联动；问题等级分类与升级机制 |
| **安检统计** | 按时间/站点/项目等维度统计、趋势分析 | 报表中心可引用；支持可视化图表 |
| **检查报表** | 各类检查与巡检的定制报表、合规报表 | 支持导出；满足监管要求 |
| **查询与筛选** | 机构名称、检查任务、检查项目、检查结果、时间范围、执行人员 | 查询/重置按钮；支持组合条件查询 |

### 2. 能源交易

#### 2.1 价格管理

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **实时调价** | 按枪/按品号调价，立即或定时生效、调价历史记录 | 站端显示「单价」如 6.05；支持批量调价 |
| **调价防御** | 单次调价幅度上限、审批流（可选）、操作日志、二次确认 | 风控与审计要求；敏感操作二次确认 |
| **价格公示** | 站端大屏、移动端、LED屏价格同步展示 | 与监管要求一致 |
| **阶梯/分时** | 阶梯定价、分时电价（若含充电）、会员专享价 | 按业务需要配置；支持多种定价策略 |
| **价格协议** | 大客户专属价格协议、合同价管理 | 与企业管理联动 |

#### 2.2 订单与交易处理

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **充装记录** | 按枪号、燃料、数量、金额、时间、车牌、会员等展示与查询 | 站端「充装记录」+ 智能搜索（枪号-金额/数量）；实时数据采集 |
| **订单流** | 实时订单状态（加注中、待支付、已支付、异常）、订单号生成规则 | 与支付弹窗、合并/拆单联动；订单唯一性 |
| **支付方式** | 现金、银联、微信、支付宝、会员卡、储值卡、IC卡、线下挂账、ETC、其它 | 支付弹窗内多方式选择，支持「合并支付」「拆单支付」；支付成功回调与确认 |
| **支付处理** | 订单金额、优惠、总计、结账/取消、快捷键（Enter/Esc）、小票打印 | 订单编号、枪号、商品、单价、数量、金额、标签/备注；支付倒计时 |
| **异常订单** | 掉电/未付订单拦截、挂起、补单与审核、异常原因记录 | 与风控、对账衔接；补单操作权限控制 |
| **退款与补单** | 退款申请与审核、补单记录与原因、退款流水 | 操作溯源；退款审批流程 |
| **标签/备注** | 订单关联标签（如「滴滴(6.05)」）、备注信息、订单分类 | 便于大客户对账与统计；自定义标签 |
| **订单详情** | 订单完整信息、支付明细、优惠明细、会员信息、车辆信息 | 支持订单详情页查询与打印 |
| **订单导出** | 按条件导出订单Excel/PDF | 财务对账、报表生成 |

#### 2.3 库存管理（后台一级模块）

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **进销存** | 卸车记录、采购入库、出库（销售/站耗）、日结盘点、月度盘点 | 与订单、罐存联动；库存流水记录 |
| **罐存比对** | 理论罐存与实测液位比对、差异分析、盘盈盘亏 | 路耗/站耗分析基础；自动触发预警 |
| **路耗/站耗** | 运输与站内损耗统计与报表、损耗率分析 | 可选：与承运商、供应商对账；异常损耗预警 |
| **库存预警** | 油品/气品库存低位预警、商品库存预警、发票库存预警 | 阈值可配置；自动通知相关人员 |
| **入库管理** | 采购入库单、供应商信息、入库数量、质量检测 | 与供应链管理、供应商对账联动 |
| **出库管理** | 销售出库、站耗出库、调拨出库、出库审核 | 出库单生成与审批 |
| **临期提醒** | 商品临期提醒、批次管理 | 减少损耗，促进销售 |

### 3. 会员与大客户

#### 3.1 会员管理

| 二级功能 | 功能点 | 说明与补充 |
|----------|--------|------------|
| **会员档案** | 注册、手机号/IC卡/二维码标识、等级、标签、生命周期 | 站端支持「会员查询」「会员管理」「添加会员」；会员唯一性校验 |
| **会员注册与认证** | 手机号注册、微信快捷注册、会员实名认证、会员信息完善 | 注册渠道追踪；实名认证与安全 |
| **会员查询** | 按手机号/IC卡/二维码/车牌快速查会员信息 | 支付侧边栏「请输入指定积分人员手机号码」；模糊查询支持 |
| **会员档案详情** | 基本信息(姓名、手机、性别、生日等)、车辆信息绑定(车牌号、车型、车辆颜色)、会员标签管理、会员分级(普通会员、VIP、SVIP等) | 会员360度视图：消费记录、积分、储值、优惠券、订单、发票 |
| **等级与权益** | 等级规则、升降级规则、专属权益（折扣、积分倍率等）、生日特权、专属折扣 | 与智慧营销联动；等级自动升降 |
| **积分归属** | 指定积分人员（本次消费积分归属哪个会员）、积分转赠 | 支付时可选会员绑定订单；积分共享规则 |
| **会员标签** | 会员标签体系、自动打标、手动打标 | 用于精准营销 |
| **会员生命周期** | 新会员、活跃会员、沉睡会员、流失会员识别与召回 | RFM模型分析支持 |

#### 3.2 大客户管理（企业车队）（截图：企业信息）

| **系统配置** | 大客户模块相关参数（欠款额度、结算周期等） | 可放在系统权限或独立配置 |

## 八、PRD 编写时可重点考虑的扩展点

1. **多业态统一账号与支付**：加气、充电、便利店、洗车是否统一会员与支付（一卡/一码全站通行）  
2. **离线与弱网**：站端断网时订单本地落库、联网后自动同步与对账策略  
3. **大客户开放能力**：是否提供 Open API（加气轨迹、账单、对账），供 LNG 承运商等 B 端对接  
4. **合规与国标**：新税控、电子发票、安全巡检等政策与国标响应速度与配置化程度  
5. **设备与物联网**：加气机、储罐、充电桩、传感器等接入方式、协议与运维监控深度
6. **实体卡与电子卡互通**：保留实体卡和实体卡管控系统,原有流程不变；系统具备实体卡绑定功能；基于加机后台管理系统的加气卡号自动查询绑定会员，自动将订单标记为会员单，自动结算为IC卡支付；营销活动可以正对实体卡进行消费后营销，如加注满1000即可赠送餐饮券、小商品、充值返利券、抽奖等
7. **异业合作生态**：与4S店、洗车店、维修店、餐饮店等异业合作，相互引流、相互营销、激活市场、资源共享
