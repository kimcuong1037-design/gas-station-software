# LNG加气站核心人员职能及协作流程

该文章详细梳理了 **LNG加气站** 的专项人员配置与协作流程。LNG站点因涉及低温液体（-162℃）和高压气化工艺，其安全要求、技术复杂度和人员资质要求均高于常规CNG站。

---

## 一、LNG加气站核心人员配置

### 1. 管理层

| 角色                    | 核心职责                             | LNG专项要求                                              |
| ----------------------- | ------------------------------------ | -------------------------------------------------------- |
| **站长**                | 全面运营、政府对接、应急管理         | 需持《危险化学品经营许可证》负责人资格证、压力容器管理证 |
| **技术负责人/生产经理** | 工艺管理、技术方案制定、疑难故障处理 | 需化工/深冷专业背景，持压力管道/压力容器管理证           |

### 2. 安全与工艺岗位（LNG核心）

| 角色                | 核心职责                                     | 资质要求                                             |
| ------------------- | -------------------------------------------- | ---------------------------------------------------- |
| **安全主管/安全员** | 安全制度、隐患排查、应急演练、事故调查       | 危险化学品安全管理人员证、注册安全工程师（优先）     |
| **工艺技术员**      | LNG工艺流程监控、BOG（蒸发气）管理、能耗优化 | 化工/深冷专业，持压力容器操作证                      |
| **设备工程师**      | 低温泵、空温式气化器、储罐、加液机维护       | 持特种设备作业证（压力容器、压力管道）               |
| **深冷维修工**      | 低温阀门、真空管道、潜液泵检修               | 需低温作业经验，持特种作业证（制冷与空调或压力容器） |

### 3. 关键操作岗位（LNG特有）

| 角色                  | 核心职责                              | 作业特点                                   |
| --------------------- | ------------------------------------- | ------------------------------------------ |
| **卸液操作员**        | LNG槽车卸液操作、卸车臂连接、储罐进液 | **核心高危岗位**，需持移动式压力容器充装证 |
| **加液操作员**        | 车辆加液、回气操作、预冷管理          | 需持气瓶充装证，掌握低温防护知识           |
| **BOG回收操作员**     | 蒸发气回收系统监控、再液化或外输操作  | 部分大型站单设，中小站由卸液员兼管         |
| **槽车引导员/安检员** | 进站槽车证件查验、车体检查、静电消除  | 需熟悉LNG槽车结构、紧急切断系统            |

### 4. 辅助岗位

| 角色                 | 职责                                | 备注                                |
| -------------------- | ----------------------------------- | ----------------------------------- |
| 财务人员/统计员      | 气款结算、成本核算、LNG库存管理     | 需理解LNG体积/质量换算（如Nm³与kg） |
| 质检员（部分站单设） | LNG组分检测、热值计算、质量纠纷处理 | 需使用色谱分析仪                    |
| 消防值班员           | 消防系统监控、泡沫/干粉系统维护     | 24小时值班，需建（构）筑物消防员证  |

## 二、LNG加气站关键协作流程

### 1. 卸液作业流程（最高风险环节）

```
【卸液前准备】（30-60分钟）
槽车到站 → 引导员查验证件（五证一单）→ 引导至卸车区
                ↓
      安全员现场确认：车轮固定、静电接地、防火间距
                ↓
      卸液操作员检查：槽车压力、液位、阀门状态、紧急切断阀
                ↓
      工艺技术员确认：储罐液位≤85%、压力正常、BOG系统就绪
                ↓
      连接卸车臂（液相+气相）→ 预冷管线（小流量进液）→ 开始卸液

【卸液过程监控】（1.5-3小时）
卸液操作员：监控槽车与储罐压差、流量、温度
                ↓
      工艺技术员：调节BOG压缩机/再液化系统，控制储罐压力
                ↓
      安全员：现场巡查，监测可燃气体浓度（<25%LEL）
                ↓
      消防值班员：消防系统处于自动状态

【卸液完成】
卸液操作员：吹扫管线（氮气或BOG）→ 拆卸卸车臂 → 复位阀门
                ↓
      槽车静置10分钟 → 引导员确认无泄漏 → 车辆离站
                ↓
      双方签认《卸液单》→ 财务核对数量（地磅/流量计双确认）
```

### 2. 加液作业流程

```
【车辆进站】
引导员：引导至加液位 → 查验车用气瓶使用登记证、年检标签
                ↓
      加液操作员：检查加液枪、回气枪、车载瓶压力/真空度
                ↓
      发现异常 → 立即停止 → 报告安全员 → 启动应急预案

【加液操作】
加液操作员：连接静电接地 → 连接回气枪（平衡压力）→ 连接加液枪
                ↓
      预冷加液机/管路（小流量）→ 正常加液（大流量）
                ↓
      监控：车载瓶液位≤90%、加液枪无结霜异常、无泄漏
                ↓
      加液完成 → 吹扫加液枪（防止冰堵）→ 拆卸枪头 → 结算

【应急处置】（关键协作）
若发生泄漏/火灾：
加液操作员 → 立即停止作业 → 按下ESD紧急切断按钮
                ↓
      安全员 → 启动警报 → 组织人员疏散 → 启动消防系统
                ↓
      站长 → 上报公司/消防/应急局 → 现场指挥
                ↓
      工艺技术员 → 隔离事故区域 → 控制BOG排放 → 防止翻滚（Rollover）
```

### 3. BOG（蒸发气）管理流程（LNG特有）

```
储罐自然蒸发 → BOG压缩机增压 → 优先：再液化回收
                ↓
      若再液化故障 → 切换至：外输管网/火炬燃烧/槽车回收
                ↓
      工艺技术员监控：储罐压力<1.2MPa（设计值因站而异）
                ↓
      压力异常升高 → 启动安全阀 → 安全员记录事件 → 站长上报
```

### 4. 设备维护协作流程

| 维护类型     | 协作流程                                                     | LNG特殊要求                            |
| ------------ | ------------------------------------------------------------ | -------------------------------------- |
| **日常巡检** | 操作员每2小时巡检 → 记录储罐液位/压力/真空度 → 异常报技术员  | 重点查真空管道外壁结霜（真空失效征兆） |
| **月度维护** | 技术员制定计划 → 维修工执行 → 安全员监护 → 站长验收          | 深冷阀门、潜液泵轴承检查               |
| **年度大修** | 外聘专业队伍 + 本站技术员配合 → 置换/惰化 → 动火作业票       | 储罐抽真空检测、安全阀校验             |
| **应急抢修** | 站长启动预案 → 工艺隔离 → 维修工带压堵漏（若可行）→ 全员警戒 | 必须使用防爆工具，穿戴防冻服           |

## 三、LNG站人员配置参考（按规模）

| 站型              | 日加液量   | 建议人数 | 关键岗位配置                                          |
| ----------------- | ---------- | -------- | ----------------------------------------------------- |
| **小型LNG站**     | <20吨/日   | 12-16人  | 站长1、技术兼安全1、卸液兼加液4（轮班）、财务1、维修1 |
| **中型LNG站**     | 20-50吨/日 | 20-28人  | 单设安全主管、工艺技术员、卸液班/加液班分离           |
| **大型LNG枢纽站** | >50吨/日   | 35-50人  | 设BOG回收专职岗、质检岗、多班制卸液组、专职消防队     |

## 四、LNG站关键管理要点

| 管理维度       | 核心要求                                                  |
| -------------- | --------------------------------------------------------- |
| **资质合规**   | 所有涉低温、压力容器作业人员必须持证（市监局/应急局双证） |
| **低温防护**   | 配备防冻服、防冻手套、面罩，操作区设置紧急冲淋/洗眼器     |
| **防翻滚管理** | 不同密度LNG分层监测，强制循环或翻滚预警系统               |
| **真空管理**   | 储罐、管道真空度年度检测，外壁结霜立即停用检修            |
| **信息化**     | SCADA系统监控储罐液位/压力/温度，联锁ESD系统              |
